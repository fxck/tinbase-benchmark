A shareable live demo on a `SERIOUS` (dedicated) core with a public subdomain and
2–16 GB / 2–8 vCPU vertical autoscaling. This is the ceiling for this app: one
strong container, because the in-process engine cannot be spread across many. For
a horizontally-scaled backend, use a managed `postgresql` service instead.
